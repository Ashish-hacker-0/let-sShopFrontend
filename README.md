<h2>Functionality and Features</h2>
<ul>
  <li>JWT authentication</li>
  <li>Admin Panel to upload products, manage Users and Check Orders.</li>
  <li>Search Result</li>
  <li>Search Filter</li>
  <li>Add To Cart</li>
  <li>Checkout</li>
  <li>User Dashboard to see orders and update Profile.</li>
</ul>
